There are two options to block Discord on iPhone (or iPad):

  • Option 1: Use the built-in Apple Screen Time feature
  • Option 2: Install an app designed to block apps (Upshift)

Option 1: Apple Screen Time

Choose this option if you want to avoid installing another application. However, you lose powerful features and flexibility.

Steps to block Discord on iPhone (or iPad) using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Select "Screen Time"
  3. Select "App Limits"
  4. Select "Add Limit"
  5. Select the app(s) to time limit(Discord)
  6. Select "Next," set the time limit, and select "Add"

Select "Add Limit"

Select The App To Limit

Select "Next," Set Time Limit, And Select "Add"

The App Will Now Be Blocked After The Time Limit Is Reached

Option 2: Upshift

The best option to block Discord if you are willing to install another application.

Steps to block Discord on iPhone (or iPad) using Upshift:

  1. Install "Upshift" from the Apple App Store
  2. Open "Upshift" app
  3. Complete the onboarding steps.

Install Upshift

Open Upshift and Go to "Focus" Page

Open App Blocker

Now You Have more Options: use any of the pre-built blocks, use quest blocker, block now or time limit

Use block now, if you want to block apps only for a few hours

Use time limit to use apps only for a specific duration

Use quest blocker, if you want to block apps until you complete your quests

Use pre-built blocks, like "Work Schedule", "weekend detox", "Games", "Porn Block", etc.
