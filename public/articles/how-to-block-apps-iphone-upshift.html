How To Block Apps on iPhone [2025]

There are two primary options to block apps on iPhone:

  • Option 1: Use the built-in Apple Screen Time feature
  • Option 2: Install an app designed to block apps; Upshift - app blocker for iPhone

Option 1: Apple Screen Time

Choose this option if you want to avoid installing a blocking app. However, you lose powerful features and flexibility.

Steps to block apps on iPhone using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Press "Screen Time"
  3. Press "Downtime"
  4. Turn on "Scheduled" - this will automatically block all apps at the selected schedule except the "Always Allowed" ones

Additionally, you can also set "Time Limits" on apps by following specific directions.

Open "Settings" App

Press "Screen Time"
