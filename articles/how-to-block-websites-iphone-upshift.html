How To Block Websites on iPhone (Safari or Chrome)

There are two primary options to block websites on iPhone:

  • Option 1: Use the built-in Apple Screen Time feature
  • Option 2: Install an app designed to block websites: Upshift website blocker

Option 1: Apple Screen Time

Choose this option if you want to avoid installing another application. However, you lose powerful features and flexibility.

There are two variants to choose from:

  • Variant A (works on Safari & third-party browsers like Chrome): block specific websites while also blocking inappropriate websites
  • Variant B (only for Safari): set time limits on specific websites

Variant A is best for a child's iPhone as it automatically blocks adult websites permanently.

Apple Screen Time: Variant A (Safari, Chrome or Others)

Choose this option if you want an option that works for both Safari and third-party browsers like Chrome. This option will also automatically block inappropriate websites.

Steps to block inappropriate websites on iPhone using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Select "Screen Time"
  3. Select "Content & Privacy Restrictions"
  4. Select "App Store, Media, Web, & Games"
  5. Select "Web Content"
  6. Select "Limit Adult Websites" (you will be able to select custom websites next)
  7. In the "NEVER ALLOW:" section, select "Add Website" (add your own websites here)

Apple Screen Time: Variant B (Safari Only)

Steps to block websites on iPhone using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Select "Screen Time"
  3. Select "App Limits"
  4. Scroll to bottom and select "websites"
  5. Select "Add Website" and type in the website you want to block
  6. Select "Next" and a time of "1 min" and select "Add"
