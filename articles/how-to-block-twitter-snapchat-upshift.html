How To Block YouTube on iPhone (or iPad)

There are two options to block YouTube on iPhone (or iPad):

  • Option 1: Use the built-in Apple Screen Time feature
  • Option 2: Install an app designed to block apps (Upshift)

Option 1: Apple Screen Time

Choose this option if you want to avoid installing another application. However, you lose powerful features and flexibility.

Steps to block YouTube on iPhone (or iPad) using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Select "Screen Time"
  3. Select "App Limits"
  4. Select "YouTube" Under The "Entertainment" Category (or search "YouTube" in the search bar by scrolling down)
  5. Select "Next" and a time of "1 min" and select "Add"

Option 2: Upshift

The best option to block YouTube if you are willing to install another application.

Steps to block YouTube on iPhone (or iPad) using Upshift:

  1. Install "Upshift" from the Apple App Store
  2. Open "Upshift" app
  3. Complete the onboarding steps. The onboarding will show you how to block apps!
