How To Set Time Limits on iPhone [2025]

There are two options to put time limits on iPhone:

  • Option 1: Use the built-in Apple Screen Time feature
  • Option 2: Install an app designed to set screen time limits; Upshift - an app to limit screen time

Option 1: Apple Screen Time

Choose this option if you want to avoid installing another application. However, you lose powerful features and flexibility.

Steps to set time limits on iPhone using Apple Screen Time:

  1. Open Apple "Settings" app
  2. Select "Screen Time"
  3. Select "App Limits"
  4. Select "Add Limit"
  5. Select the app(s) to time limit
  6. Select "Next," set the time limit, and select "Add"

Option 2: Upshift

The best option to set time limits if you are willing to install another application.

Steps to set time limits on apps using Upshift:

  1. Install "Upshift" from the Apple App Store
  2. Open "Upshift" app
  3. Complete the onboarding steps. The onboarding will show you how to set time limits!
  4. Select "Start" at the bottom
  5. Select "Set Time Limits"
  6. Press "Select Apps & Websites To Time Limit"
  7. Choose the apps to time limit (for example, you can use Upshift as an app to limit social media)
  8. Select the time limit duration (ex. 5 minutes)
  9. Press "Set Time Limit"
